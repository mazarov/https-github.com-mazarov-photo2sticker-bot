## 4. Убрать кнопку «Праздник»

Кнопка Holiday (`🎄 8 марта: вкл/выкл`) убирается из:
- `showStickerIdeaCard` (идея стикера)
- `buildPackCarouselCard` (карусель паков)

**Причина:** праздничный фильтр/тег применяется автоматически по дате. Кнопка добавляла когнитивную нагрузку и занимала ряд в клавиатуре. Пользователю не нужно вручную включать/выключать праздник.

# Пакет "Попробовать" (Try) — дешёвый вход для новых пользователей

**Дата:** 2026-02-12
**Статус:** Реализовано

---

## Проблема

Минимальный пакет — 10 кредитов за 150 Stars. Для нового пользователя это высокий порог входа. Нужен дешёвый пробный пакет.

## Решение

Новый пакет "Попробовать" / "Try":
- **2 кредита** за **20 Stars** (~15₽)
- Виден **только** пользователям с `has_purchased = false`
- После первой покупки пакет исчезает из меню
- Защита от повторной покупки через прямой callback

## Также

- Переименован пакет "Популярный" → "Поп" / "Popular" → "Pop" (короче, лучше влезает в кнопку)

## Изменения

**Файл:** `src/index.ts`

1. `CREDIT_PACKS` — добавлен `{ credits: 2, price: 20, trialOnly: true }`
2. `sendBuyCreditsMenu` — фильтр `!p.trialOnly || !user.has_purchased`
3. `pack_select` handler — блокировка trialOnly при `user.has_purchased`

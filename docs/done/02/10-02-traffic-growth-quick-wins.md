## Текущая ситуация

- 2 ключевых слова в Яндекс Директ

### UTM-разметка (динамические параметры Яндекс Директ)

Все ссылки в объявлениях размечаются динамическими параметрами Директа.
При клике Яндекс подставит реальные значения вместо `{...}`.
Справка: https://yandex.ru/support/direct/ru/statistics/url-tags

**Шаблон основной ссылки объявления:**
```
https://photo2sticker.ru/?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content={ad_id}.{position_type}.{position}&utm_term={keyword}
```

**Шаблон быстрых ссылок (sitelinks):**
```
https://photo2sticker.ru/PAGE?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content=sitelink_N.{ad_id}&utm_term={keyword}
```

**Шаблон deep link на бота (start payload):**
```
https://t.me/photo2sticker_bot?start=ya_{campaign_id}_{ad_id}
```

> ⚠️ В быстрых ссылках параметры `{campaign_id}`, `{ad_id}`, `{phrase_id}` подставляются,
> только если эти же параметры есть в основной ссылке объявления.

| Параметр | Что подставится |
|----------|----------------|
| `{campaign_id}` | ID кампании (число) |
| `{ad_id}` | ID объявления (число) |
| `{keyword}` | Ключевая фраза показа |
| `{position_type}` | Тип блока: premium / dynamic_places / other / none |
| `{position}` | Позиция в блоке (1, 2...) |
| `{source}` | Домен площадки РСЯ или none (поиск) |
| `{source_type}` | search / context |
| `{device_type}` | desktop / mobile / tablet |
| `{region_id}` | ID региона показа |

## Шаг 1.5: Оптимизация формата объявлений

### Проблемы текущего формата

Типичные ошибки в объявлениях Яндекс Директ для данной ниши:

1. **Заголовок 1 не содержит ключевую фразу** — снижает релевантность и CTR. Яндекс подсвечивает жирным совпадения с запросом пользователя
2. **Заголовок 2 не используется или пустой** — теряем 30 символов бесплатного текста
3. **Нет конкретики в тексте** — "создание стикеров" без цифр, сроков, выгоды
4. **Нет расширений** — быстрые ссылки, уточнения, отображаемая ссылка = бесплатные клики по площади
5. **Одно объявление на группу** — нет A/B тестирования, нет адаптации под запрос
6. **Нет "бесплатно" в объявлении** — главный триггер ниши (ТОП-запросы содержат "бесплатно")

### Лимиты Яндекс Директ (текстово-графическое объявление)

| Элемент | Лимит символов | Обязательный |
|---------|---------------|-------------|
| Заголовок 1 | 56 | ✅ |
| Заголовок 2 | 30 | Нет, но НАДО |
| Текст объявления | 81 | ✅ |
| Отображаемая ссылка | 20 | Нет, но НАДО |
| Быстрые ссылки | 4 шт × 30 симв + описание | Нет, но НАДО |
| Уточнения | 25 симв × до 4 шт | Нет, но НАДО |

**Правило:** заполнять ВСЕ поля — каждое расширение увеличивает площадь объявления в выдаче и CTR на 15-25%.

### Рекомендации по A/B тестированию объявлений

| Что тестировать | Вариант A | Вариант B |
|----------------|-----------|-----------|
| Триггер | "Бесплатно" | "За 30 секунд" |
| Формат | Шаги (Отправь→Выбери→Получи) | Выгоды (ИИ, 10 стилей, без регистрации) |
| CTA в тексте | "Попробуй сейчас!" | "Попробуй бесплатно!" |
| Числа | "50 000+ стикеров создано" | "10+ стилей на выбор" |
| Заголовок 2 | Сроки ("Готово за 30 секунд") | Бесплатность ("Первый бесплатно") |

**Стратегия:** по 2 объявления на группу, ротация "Оптимизировать по конверсиям", оценка через 500-1000 показов на объявление.

### Чек-лист формата объявлений

- [ ] Заголовок 1 содержит ключевую фразу (для подсветки в выдаче)
- [ ] Заголовок 2 заполнен (конкретная выгода или CTA)
- [ ] Текст содержит: выгоду + конкретику (цифры) + CTA
- [ ] Отображаемая ссылка — кириллица, релевантна запросу
- [ ] 4 быстрые ссылки с описаниями
- [ ] 4 уточнения (USP: бесплатно, скорость, стили, без регистрации)
- [ ] 2 объявления на группу для A/B теста
- [ ] Слово "бесплатно" есть в заголовке или тексте

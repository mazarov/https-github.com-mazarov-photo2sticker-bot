## 2. Структура кампании (рекомендуемая)

- **Вариант A:** Одна кампания «Photo2Sticker — пак стикеров», 2–3 группы по типу запроса/хука (универсальные, бесплатно, бот).
- **Вариант B:** Несколько кампаний по темам (например: «Стикерпак из фото», «Бесплатно», «Бот») — для раздельного контроля бюджетов и ставок.

В каждой группе — по 3–5 объявлений с разными заголовками (хуками) для теста.

## 3. Объявления: параметры по типам

Формат объявления Яндекс.Директ (поиск): **Заголовки** (до 4, лимиты символов по правилам Директа на момент запуска), **Текст объявления**, **Отображаемая ссылка**, **Быстрые ссылки** (до 4).

Ниже — контент для подстановки. Лимиты: заголовок обычно до 56 символов с пробелами, текст — до 81 символа в одной строке, общая длина текста — по интерфейсу Директа.

### 3.2. Текст объявления (body)

Варианты основного текста (подставить в объявление, при необходимости разбить на строки по лимиту):

1. **Универсальный:**

   (отображаемая ссылка: photo2sticker.ru → например «Перейти на сайт» или «Сделать пак»)

2. **Бесплатно:**  
   Первый пак стикеров из фото — бесплатно. В Telegram. Без регистрации. За 30 секунд.

3. **Бот:**  
   Telegram-бот: пришли фото → выбери стиль и эмоции → получи пак стикеров. Первый пак бесплатно.

4. **Простота:**  
   Пак стикеров из фото без редакторов и дизайнера. ИИ сохраняет черты лица. 30 секунд — готово.

5. **Интрига:**  
   Что будет, если отправить фото боту в Telegram? Готовый пак стикеров с разными эмоциями за 30 сек.

**Отображаемая ссылка (видимый URL):**  
- Вариант 1: photo2sticker.ru  
- Вариант 2: photo2sticker.ru/пак-стикеров (если есть такой путь)  
- Путь не должен быть длинным; цель — доверие и узнаваемость.

### 3.3. Быстрые ссылки (до 4 штук)

Назначение: развести трафик по смыслу и при необходимости по UTM. **Обязательно прокидывать:** source, medium, campaign (подстановка Директа), что это блок быстрых ссылок (utm_content=sitelinkN_…), запрос пользователя (utm_term={keyword}).

В быстрых ссылках макросы `{campaign_id}` и `{keyword}` подставляются Директом только если они есть в **основной ссылке объявления** — см. п. 3.4.

| Текст быстрой ссылки | URL (с UTM) | Комментарий |
|----------------------|-------------|-------------|
| Сделать пак стикеров | `?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content=sitelink1_pack&utm_term={keyword}` | Основной переход; content = блок быстрых ссылок, 1-я ссылка |
| Как это работает | `?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content=sitelink2_how&utm_term={keyword}` | content = 2-я быстрая ссылка |
| Открыть бота | `?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content=sitelink3_bot&utm_term={keyword}` | content = 3-я быстрая ссылка |
| Цены | `?utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content=sitelink4_faq&utm_term={keyword}` | content = 4-я быстрая ссылка |

Полный URL: `https://photo2sticker.ru/` + строка из таблицы.

Альтернативные формулировки текста быстрых ссылок:
- «Сделать пак стикеров»
- «Пак за 30 секунд»
- «Бот в Telegram»
- «Цены и условия»

Длина текста быстрой ссылки — по лимиту Директа (обычно до 30 символов).

На лендинге блоки «Боль» и «Надежда» совпадают с этими ожиданиями; после клика пользователь попадает в ту же логику воронки.

## 5. UTM-разметка

Единый формат для всех объявлений и быстрых ссылок. **Прокидывать обязательно:**

| Параметр   | Значение / макрос | Что даёт на сайте |
|------------|-------------------|-------------------|
| utm_source | yandex            | Источник — Яндекс |
| utm_medium | cpc               | Канал — контекст |
| utm_campaign | {campaign_id}   | Номер кампании в Директе (подстановка) — из какой кампании пришёл пользователь |
| utm_content | см. ниже         | Тип клика: основная ссылка / быстрая ссылка (какая именно) / кнопка |
| utm_term   | {keyword}         | Запрос пользователя (подстановка Директа) — по какому ключу пришёл |

**Основная ссылка объявления:**  
`utm_content={ad_id}.{position_type}.{position}` (или свой шаблон).

**Быстрые ссылки (блок быстрых ссылок):**  
`utm_content=sitelink1_pack`, `sitelink2_how`, `sitelink3_bot`, `sitelink4_faq` — по нему видно, что клик с быстрых ссылок и какая именно.

**Кнопка действия:**  
`utm_content=button`.

Макросы `{campaign_id}` и `{keyword}` в быстрых ссылках подставляются только если они есть в основной ссылке объявления (требование Директа).

На лендинге эти UTM должны передаваться в кнопку/ссылку в бота (например в `start`-параметр), чтобы в боте видеть источник трафика.

## 7. Контроль и доработки

- После запуска: смотреть отчёт по поисковым запросам, дополнять минус-слова, отключать нецелевые.
- A/B: сравнивать группы и объявления с разными заголовками (хуками); оставлять лучшие по CTR и конверсии в переход в бота.
- Лендинг: для кампаний с utm_campaign=free, bot, ai, telegram проверить, что на сайте подставляются нужные H1/sub (см. ТЗ на лендинг).
